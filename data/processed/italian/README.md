# data/processed/italian

Preprocessed audio files and manifest CSV for the Italian Speech Dataset.
All files here are generated by the preprocessing pipeline — never edit manually.

## Contents (after preprocessing)

```
data/processed/italian/
├── {subject_id}_{task}_preprocessed.wav   # Resampled, denoised, normalised audio
└── italian_manifest.csv                   # Dataset manifest (generated)
```

Actually the manifest is stored one level up as `data/processed/italian_manifest.csv`.

## Regenerating

```bash
python scripts/preprocess.py --dataset italian --config configs/svm_italian_mfcc.yaml
```

## DVC Tracking

```bash
dvc add data/processed/italian
git add data/processed/italian.dvc
git commit -m "data(italian): add preprocessed recordings v1.0"
dvc push
```
