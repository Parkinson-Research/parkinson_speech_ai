# data/processed/mdvr_kcl

Preprocessed audio files and manifest CSV for the MDVR-KCL Dataset.
All files here are generated by the preprocessing pipeline — never edit manually.

## Contents (after preprocessing)

```
data/processed/mdvr_kcl/
└── {subject_id}_{task}_preprocessed.wav   # 16 kHz mono, denoised, normalised
```

The manifest is stored one level up as `data/processed/mdvr_kcl_manifest.csv`.

## Regenerating

```bash
python scripts/preprocess.py --dataset mdvr_kcl --config configs/xgboost_mdvrkcl_egemaps.yaml
```

## DVC Tracking

```bash
dvc add data/processed/mdvr_kcl
git add data/processed/mdvr_kcl.dvc
git commit -m "data(mdvr-kcl): add preprocessed recordings v1.0"
dvc push
```
